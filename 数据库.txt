│ homework     │ 课后作业（科目、截止时间、附件）                │
│ activity     │ 学生活动（时间、地点、状态）                    │
│ activity_image│ 活动图片（多图上传）                          │
│ activity_comment│ 活动留言（支持回复）                        │
│ grade        │ 成绩表（排名、平均分）                         │
│ reward_punishment│ 奖罚记录                                  │
│ evaluation   │ 教师评语                                      │
│ attendance   │ 考勤记录（进/出校、数据来源）                   │
